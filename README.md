# 🧠 CSharpExamples

Bu depo, C# diliyle yapılmış çeşitli örnek projeleri içermektedir.  
Her klasör, farklı bir konuyu veya uygulama örneğini temsil eder.

## 📂 Proje Listesi

- **alistirma1** → Konsol uygulaması temel örneği  
- **alistirma2** → Dizilerle çalışma  
- **alistirma3** → Dosya işlemleri  
- **alistirma4** → OOP (Nesne yönelimli programlama) örneği  
- **alistirma5** → Koleksiyonlar ve LINQ  

## ⚙️ Kullanım
Projeyi çalıştırmak için:
1. İstediğiniz klasörü Visual Studio ile açın  
2. `Program.cs` dosyasını çalıştırın  
3. Konsoldaki çıktıyı inceleyin
